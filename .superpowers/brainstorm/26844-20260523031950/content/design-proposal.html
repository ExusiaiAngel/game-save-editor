视觉设计系统

要素方案
底色#0d1117 (主背景) / #161b22 (卡片) / #21262d (悬浮)
强调色#58a6ff (蓝) 主色 / #3fb950 (绿) 成功 / #d29922 (黄) 警告
字体Consolas / Cascadia Code 作为主等宽字体，Segoe UI 用于标题
图标保留 Unicode 表情符号，但统一风格（↗替换 ▶ 等）
圆角卡片 6px / 按钮 4px / 输入框 4px
间距模块间距 12px / 内边距 8px / 紧凑但不拥挤
暗色切换提供 🌙/☀ 一键切换，默认暗色

优点

  • 模块化导航，结构清晰可扩展
  • 每个视图独占全宽，字段列表不再拥挤
  • 仪表盘提供统一入口，降低上手门槛
  • 备份管理独立成视图，更符合专业工具定位
  • 暗色游戏主题，符合目标用户审美

风险

  • 存档编辑和实时修改分成两个标签，跨面板拷贝需重新设计
  • egui 没有原生"router/导航"概念，需自行管理视图状态
  • 侧栏宽度需权衡（太窄看不清、太宽挤占内容区）
